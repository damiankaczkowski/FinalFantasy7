1) Create a folder called C:\Program Files\Square Soft, Inc\Final Fantasy VII\movies
2) Insert Final Fantasy VII CD1, copy all files from the movie folder to C:\Program Files\Square Soft, Inc\Final Fantasy VII\movies
3) repeat for each disk, so all movies from all disks are on your hard drive in the same movies folder
4) Click Start, Run, type regedit, push OK
5) Go to HKEY_LOCAL_MACHINE\SOFTWARE\Square Soft, Inc.\Final Fantasy VII
6) Double-click on MoviePath and change it from "d:\FF7\Movies\" to C:\Program Files\Square Soft, Inc\Final Fantasy VII\movies\
7) Push OK and close regedit
